## 내가 푼 풀이법
### 슈도코드
```
person(3명의 수포자들의 점수판)
p1(1번 수포자가 찍는 방식)
p2(2번 수포자가 찍는 방식)
p3(3번 수포자가 찍는 방식)

# 입력받은 answers와 수포자 찍는 방식 비교해서 점수내기
for answers
    if 1번이 찍은 방식과 answers가 일치할때 1번 점수
    if 2번이 찍은 방식과 answers가 일치할때 2번 점수
    if 3번이 찍은 방식과 answers가 일치할때 3번 점수

max_person(최대 점수)
# 점수판과 최대점수를 비교해서 최댓값을 받은 사람을 리스트 반환
for person
    if 최대점수와 점수판 점수가 같을때
        answer.append(수포자번호)
```
### 코드 구현
```
def solution(answers):
    answer = []
    person = [0,0,0]  # 점수
    p1 = [1,2,3,4,5]
    p2 = [2,1,2,3,2,4,2,5]
    p3 = [3,3,1,1,2,2,4,4,5,5]
    
    for i in range(len(answers)):
        if answers[i] == p1[i%5]:
            person[0] += 1
        if answers[i] == p2[i%8]:
            person[1] += 1
        if answers[i] == p3[i%10]:
            person[2] += 1
    max_person = max(person)
    
    for i in range(len(person)):
        if person[i] == max_person:
            answer.append(i+1)
    return answer

```
