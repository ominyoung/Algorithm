## 내가 푼 풀이법
### 슈도코드
```
w (가로길이 리스트)
h (세로길이 리스트)

for sizes만큼 반복
    만약 세로가 가로보다 긴 명함일 경우,
    명함을 돌림(가로와 세로의 길이를 변환함)

for sizes만큼 반복
    w.append(sizes 값들의 가로)
    h.append(sized 값들의 세로)

가로의 최댓값 * 세로의 최댓값 출력
```
### 코드 구현
```
def solution(sizes):
    w = []
    h = []
    # 세로가 긴 명함 회전
    for i in sizes:
        if i[0] < i[1]:
            i[0],i[1] = i[1],i[0]
    
    for i in sizes:
        w.append(i[0])
        h.append(i[1])
    answer = max(w)*max(h)
    return answer
```
